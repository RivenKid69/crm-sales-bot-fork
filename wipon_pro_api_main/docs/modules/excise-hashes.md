# excise-hashes

Этот модуль отвечает за хранение, поиск и создание уникальных хэшей акцизных марок — специальных кодов, привязанных к товарам, подлежащим акцизным сборам. Он работает как «библиотека» для этих хэшей, позволяя другим частям системы быстро находить нужные данные или добавлять новые. Использует базу данных через TypeORM и содержит бизнес-логику для проверки корректности операций.

## Responsibilities

- Создание новых акцизных хэшей при добавлении товаров в систему
- Поиск существующих хэшей по уникальным идентификаторам
- Проверка корректности хэшей перед их использованием в бизнес-процессах
