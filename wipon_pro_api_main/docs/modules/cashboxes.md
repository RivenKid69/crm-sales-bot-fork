# cashboxes

Модуль cashboxes отвечает за управление кассами в приложении, интеграцию с уведомлениями через FCM и взаимодействие с пользователями. Он структурирует логику работы с кассами, используя сервисы и контроллеры, а также связывает систему с внешними компонентами, такими как уведомления и база данных. Модуль обеспечивает централизованное управление операциями, связанными с кассами, и координирует их с другими частями приложения.

## Responsibilities

- Управление логикой работы с кассами (создание, обновление, удаление)
- Интеграция с системой уведомлений FCM для отправки сообщений
- Связь с модулем пользователей для обработки данных, связанных с кассами
