# pdf

Этот модуль отвечает за создание, генерацию и отправку PDF-чеков и счетов-фактур пользователям. Он интегрируется с системой авторизации, использует данные пользователей из других модулей и обеспечивает безопасный API для работы с документами. Модуль работает как 'фабрика документов', которая получает данные, формирует PDF и отправляет их по запросу.

## Responsibilities

- Генерация PDF-счетов-фактур на основе данных пользователя и настроек подписки
- Обработка HTTP-запросов для создания и отправки PDF-файлов с защитой от неавторизованных доступов
- Интеграция с модулем пользователей для получения данных, необходимых для формирования документов

- бухгалтерия, документооборот, платежи

- PDF-генерация (чеки, документы)

## Dependencies

This module depends on:

- express: для обработки HTTP-запросов и отправки PDF-файлов
- fs: для работы с файловой системой при создании и сохранении PDF-файлов
- @nestjs/common: для общих функций и декораторов в NestJS

## Main Exports

- `PdfService: основной сервис для генерации PDF-документов`
- `PdfController: контроллер, обрабатывающий HTTP-запросы для создания и отправки PDF-файлов`
