# tasks

Этот модуль отвечает за выполнение фоновых задач, таких как фискализация платежей, синхронизация магазинов с лицензиями, отправка уведомлений о заканчивающихся подписках и автоматическая очистка данных. Он взаимодействует с базой данных, кэшем и внешними сервисами, обеспечивая бесперебойную работу системы в фоновом режиме.

## Responsibilities

- Автоматическая фискализация платежей через HTTP-запросы и работу с кэшем
- Синхронизация магазинов с лицензиями по расписанию с использованием TypeORM
- Отправка push-уведомлений о заканчивающихся подписках через FCM
