# dao

Этот модуль отвечает за определение и управление моделями данных, которые используются для хранения информации в базе данных. Он охватывает такие сущности, как аккаунты, транзакции, пользователи, подписки, устройства, лицензии и отчёты. Модуль использует TypeORM для взаимодействия с базой данных и обеспечивает структурированный доступ к данным, что позволяет другим частям приложения работать с информацией в безопасном и понятном виде.

## Responsibilities

- Определение моделей данных (таблиц) для хранения информации об аккаунтах, транзакциях, пользователях, подписках, устройствах, лицензиях и отчётах
- Управление связями между сущностями (например, связь между пользователями и их устройствами или подписками)
- Обеспечение работы с временными метками (updated_at) и статусами (is_active) для отслеживания изменений и состояний
