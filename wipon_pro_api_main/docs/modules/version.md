# version

Этот модуль отвечает за управление версиями приложений и предоставление информации о текущих версиях через API. Он позволяет клиентам проверять, нужно ли им обновлять настольные и мобильные приложения, а также документировать API-эндпоинты. Модуль организует логику в отдельный блок для удобства управления и повторного использования кода.

## Responsibilities

- Предоставляет HTTP-эндпоинты для получения информации о текущих версиях настольных и мобильных приложений
- Использует сервис для получения данных о версиях и преобразования их в формат, понятный клиентам
- Организует логику в отдельный модуль для упрощения управления и повторного использования кода
