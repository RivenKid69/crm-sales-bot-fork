# sms

Этот модуль отвечает за отправку SMS-кодов подтверждения пользователям в системе. Он использует внешние сервисы для отправки сообщений и внутренние настройки для обеспечения безопасности. Модуль разделён на компоненты для удобства управления и повторного использования в проекте.

## Responsibilities

- Отправка кодов подтверждения пользователям через SMS
- Интеграция с внешними сервисами для отправки сообщений
- Логирование ошибок при отправке SMS
