# logger

Этот модуль отвечает за логирование событий и ошибок в приложении. Он использует библиотеку Winston для записи логов в файлы и консоль, а также интегрирует вспомогательные функции для работы со временем. Модуль разделяет логи на два типа: ошибки (error-logger) и запросы (request-logger), что помогает быстрее находить проблемы и анализировать работу системы.

## Responsibilities

- Запись ошибок в файлы для последующего анализа
- Логирование информации о запросах (например, URL, статус ответа)
- Форматирование логов с временем и уровнем серьёзности (error, info и т.д.)
