# config

Этот модуль служит централизованным хранилищем настроек для всего приложения. Он управляет SSL-ключами, ограничениями по запросам, подключениями к базам данных, настройками подписок и другими критически важными параметрами. Все конфигурации вынесены в отдельные файлы для удобства изменения и безопасного хранения чувствительных данных.

## Responsibilities

- Управление SSL-ключами для безопасного взаимодействия с платежными системами
- Настройка ограничений на количество запросов для предотвращения перегрузки
- Централизованное хранение параметров подключения к базам данных, Redis и SMS-сервисам
