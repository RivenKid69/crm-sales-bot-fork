# guards

Этот модуль отвечает за защиту API-эндпоинтов приложения. Он использует различные 'гардеи' (guards), которые проверяют API-ключи, ограничивают доступ по IP-адресам, контролируют частоту запросов и защищают транзакции. Можно представить его как систему безопасности, которая проверяет каждый входящий запрос, чтобы убедиться, что он легитимный и безопасный.

## Responsibilities

- Проверка валидности API-ключа и статуса подписки пользователя
- Ограничение доступа к API на основе IP-адреса в зависимости от окружения
- Контроль количества запросов от одного пользователя для предотвращения злоупотреблений API
- Защита транзакций через проверку токена авторизации

## Domains

This module covers the following business domains:

- API
- api
- авторизация, API
- авторизация, api
- авторизация

## Dependencies

This module depends on:

- express для работы с HTTP-запросами
- rxjs для асинхронной обработки данных
- cache-manager для хранения и управления лимитами запросов
- @nestjs/common для интеграции с фреймворком NestJS

## Main Exports

- `CheckApiKeyGuard для проверки API-ключа и подписки`
- `IpFilterGuard для ограничения доступа по IP-адресам`
- `UserRateLimitGuard для контроля количества запросов от пользователя`
- `TransactionGuard для защиты транзакций через проверку токена`
