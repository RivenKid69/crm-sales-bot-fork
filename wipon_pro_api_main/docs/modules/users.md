# users

Этот модуль отвечает за централизованное управление пользователями в приложении. Он объединяет логику создания, поиска, аутентификации и управления токенами пользователей. Модуль использует TypeORM для взаимодействия с базой данных и предоставляет API через контроллеры для других частей системы.

## Responsibilities

- Обработка создания пользователей по номеру телефона и идентификатору
- Управление токенами и аутентификацией пользователей
- Поиск пользователей по различным критериям (токен, телефон, ID)
