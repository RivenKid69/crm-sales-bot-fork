# filters

Этот модуль отвечает за централизованную обработку всех исключений в приложении. Он логирует ошибки на сервере, чтобы разработчики могли их видеть, и возвращает пользователю понятные сообщения об ошибках. Это как «пожарный шланг» в системе — он ловит все ошибки, не даёт им «сгореть» и помогает быстро их исправить.

## Responsibilities

- Ловит все исключения, возникающие в приложении
- Логирует ошибки на сервере для анализа разработчиками
- Возвращает пользователю структурированные сообщения об ошибках
