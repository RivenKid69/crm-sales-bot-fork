# API for Wipon built with NestJS

Recommended NodeJS version - `14.17.6`

Recommended NPM version - `7.23.0`

## Technology stack

PostgreSQL version 12

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Lint

```bash
$ npm run lint
```
