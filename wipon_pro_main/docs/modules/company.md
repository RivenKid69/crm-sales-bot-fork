# company

Этот модуль отвечает за централизованное хранение и управление данными компании, лицензий и магазинов в приложении на Vue.js. Он использует Vuex для управления состоянием, взаимодействует с API для получения и обновления данных, а также предоставляет доступ к этим данным через getters. Модуль играет роль 'хранилища' для всех информации, связанной с компанией, и обеспечивает согласованность данных между различными частями приложения.

## Responsibilities

- Получение и сохранение данных компании из внешних API
- Обновление и управление состоянием компании в Vuex-хранилище
- Предоставление доступа к данным компании через getters для компонентов приложения
