# router

Этот модуль отвечает за управление навигацией в веб-приложении, защищая маршруты от неавторизованного доступа. Он использует vue-router для определения маршрутов и проверяет состояние пользователя через store, чтобы решать, куда перенаправлять. Также он использует разные layouts для разделения публичных и закрытых частей приложения, как будто это пропускной пункт в здании, где некоторые двери доступны только авторизованным людям.

## Responsibilities

- Защита маршрутов, требующих авторизации, через проверку состояния пользователя
- Организация навигации с использованием разных layouts (EmptyLayout и MainLayout)
- Интеграция с хранилищем (store) для получения информации о состоянии пользователя
