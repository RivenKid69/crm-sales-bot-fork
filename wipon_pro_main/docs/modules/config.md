# config

Этот модуль служит централизованным хранилищем настроек для приложения. Он управляет ценами, типами устройств и конфигурациями API, обеспечивая единый источник данных для других частей системы. Например, как центральный ящик в доме, где хранятся все ключи и настройки, доступные всем жителям.

## Responsibilities

- Хранение и предоставление актуальных цен на товары/услуги
- Управление типами устройств, используемых в системе
- Настройка параметров подключения к внешним API
