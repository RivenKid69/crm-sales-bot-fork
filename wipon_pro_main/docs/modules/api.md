# api

Этот модуль отвечает за взаимодействие с внешними серверами через HTTP-запросы. Он обеспечивает авторизацию, управление данными компаний, получение информации о типах магазинов, обработку подписок, платежей и генерацию PDF-счетов. Модуль также содержит общие функции для безопасного взаимодействия с API, обработки ошибок и управления аутентификацией.

## Responsibilities

- Обеспечивает авторизацию пользователей через SMS-коды и управление сессиями
- Выполняет запросы к серверу для получения и обновления данных о компаниях, счетах, транзакциях и подписках
- Генерирует и скачивает PDF-счета-фактуры для клиентов

## Domains

This module covers the following business domains:

- API
- платежи
- PDF генерация, API вызовы
- авторизация
- авторизация, пользовательские данные
- api
- API (запросы к серверу)
- платежи, API

## Dependencies

This module depends on:

- axios — библиотека для выполнения HTTP-запросов
- @kyvg/vue3-notification — для уведомлений в веб-приложении при обработке ошибок
- @/config/index — для получения конфигурационных данных

## Main Exports

- `getAccountRequest — для получения данных о счёте пользователя`
- `getTransactionsRequest — для получения информации о транзакциях`
- `updateCompanyRequest — для обновления данных компании`
- `generateInvoice — для создания и скачивания PDF-счетов`
- `buySubsRequest — для покупки подписки`
- `makeRefund — для возврата средств по подписке`
