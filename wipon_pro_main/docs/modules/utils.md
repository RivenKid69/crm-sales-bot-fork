# utils

Этот модуль содержит утилиты для работы с файлами, кодирования данных и генерации QR-кодов. Он помогает скачивать файлы из API, кодировать текст в специфические форматы и создавать временные задержки. Его используют, когда нужно обработать данные или подготовить их для передачи в другие системы.

## Responsibilities

- Скачивание файлов из ответов сервера (например, PDF или изображения)
- Кодирование текста в форматы, подходящие для QR-кодов или телекоммуникационных систем
- Создание временных задержек для синхронизации операций в приложении
