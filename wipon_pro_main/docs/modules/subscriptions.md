# subscriptions

Этот модуль отвечает за управление подписками пользователей в приложении. Он хранит данные о тарифах, обрабатывает покупки, загружает информацию с сервера и предоставляет её другим частям приложения. Работает как центральный «хранилище» для всех данных, связанных с подписками, и обеспечивает их синхронизацию между интерфейсом и сервером.

## Responsibilities

- Загружает и сохраняет данные о тарифных планах с сервера
- Обрабатывает логику покупки и активации подписок
- Предоставляет доступ к данным подписок через getters для компонентов интерфейса
