# store

Этот модуль является центральным хранилищем состояния (Vuex) для Vue-приложения. Он объединяет различные модули (авторизация, подписки, аккаунт и т.д.), обеспечивая единый доступ к данным, логике изменений и отладке. Использует initialState для инициализации значений и предоставляет инструменты для управления состоянием в компонентах приложения.

## Responsibilities

- Объединение и настройка Vuex-хранилища для управления состоянием приложения
- Инициализация дефолтного состояния с использованием конфигурационных настроек
- Предоставление доступа к данным через getters, мутации и actions для компонентов
